12 ## Dependeces
13 - [manif](https://github.com/artivis/manif);
14 - [Eigen3](http://eigen.tuxfamily.org/index.php?title=Main_Page);
15 - [cmake](https://cmake.org/);
16 - [Catch2](https://github.com/catchorg/Catch2) (only for testing).

18 ## Build the library
19 ```sh
20 git clone https://github.com/GiulioRomualdi/lie-group-controllers.git
21 cd lie-group-controllers
22 mkdir build && cd build
23 cmake ../
24 cmake --build .
25 [sudo] make install
26 ```
27 If you want to enable tests set the `BUILD_TESTING` option to `ON`.
28 
29 ## Use lie-groups-controllers in your project
30 **lie-groups-controllers** provides native CMake support which allows the library to be easily used in CMake projects. Please add in your `CMakeLists.txt`
31 ```cmake
32 project(foo)
33 find_package(LieGroupsControllers REQUIRED)
34 add_executable(${PROJECT_NAME} src/foo.cpp)
35 target_link_libraries(${PROJECT_NAME} LieGroupsControllers::LieGroupsControllers)
36 ```
